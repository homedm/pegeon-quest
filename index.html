ダウンロード(PegeonQuest 実行ファイル)

ソースコードから実行ファイルを生成する手順

  1. GitHub リポジトリをクローンまたはダウンロードします。
    git clone https://github.com/homedm/pegeon-quest pegeon-quest
  2. prjディレクトリに移動します。
    cd pegeon-quest/prj
  3. 実行ファイル生成用のスクリプトを実行します。../PegeonQuest/ディレクトリに実行用のjarファイルと実行に必要なりソースがコピーされます。
    ./build.sh
