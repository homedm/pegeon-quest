Pegeon Quest - 最強の鳩を目指して -

実行方法

  1. 次の ZIP ファイルをダウンロードしてください。
    ダウンロード (PegeonQuest)
  2. Java のバージョンは JDK 1.10.1 となっています。 PegeonQuest ディレクトリにある PegeonQuest.jar を実行します。 実行方法は各 OS のやり方に従ってください。

ソースコードから実行ファイルを生成する手順

  1. GitHub リポジトリをクローンまたはダウンロードします。
    git clone https://github.com/homedm/pegeon-quest.git pegeon-quest
  2. 実行ファイル生成用のスクリプトを実行します。PegeonQuest ディレクトリにjarファイルが生成されます。
    ./build.sh
